sort

sort(@drillTools)

Takes a list of drill tools. Each tool is a hash with the following structure:

$t->{ 'TYPE' => '(hole-finish)', 'SHAPE' => '(hole|slot)', 'TOOL_NUM' => '(#)', 'MIN_TOL' => '(#)', 'MAX_TOL' => '(#)', 'DRILL_SIZE' => '(#)', 'COUNT' => '(#)', 'FINISH_SIZE' => '(#)' }

Returns a sorted list of drill tools by drill size. In the case that two drill tools have the same drill size it falls back on finish size.

See SYNOPSIS for detailed usage.

sortBy

sortBy("NPTLast",@drillTools)

Takes a list of drill tools. Each tool is a hash with the following structure:

$t->{ 'TYPE' => '(hole-finish)', 'SHAPE' => '(hole|slot)', 'TOOL_NUM' => '(#)', 'MIN_TOL' => '(#)', 'MAX_TOL' => '(#)', 'DRILL_SIZE' => '(#)', 'COUNT' => '(#)', 'FINISH_SIZE' => '(#)' }

Returns a sorted list of drill tools by drill size. In the case that two drill tools have the same drill size it falls back on finish size. Non-plated holes will come last (sorted in the same fashion)

sortBy("NPTFirst",@drillTools) Returns a sorted list of drill tools by drill size. In the case that two drill tools have the same drill size it falls back on finish size. Non-plated holes will come first (sorted in the same fashion)

sortBy("CountASC",@drillTools) Returns a sorted list of drill tools by count in ascending order, that's tool sizes with the fewest counts first. In the case that two drill sizes have the same count it will fall back on drill size in ascending order.

sortBy("CountDSC",@drillTools) Returns a sorted list of drill tools by count in descending order, that's tool sizes with the greatest counts first. In the case that two drill sizes have the same count it will fall back on drill size in ascending order.

sortBy("ToolASC",@drillTools) Returns a sorted list of drill tools by tool number in ascending order. Note tool numbers are sometimes arbitrary, but this will be the same order as drill tools manager.

sortBy("ToolDSC",@drillTools) Returns a sorted list of drill tools by count in descending order. Note tool numbers are sometimes arbitrary, but this will be the same order as drill tools manager.

sortBy("Descending",@drillTools) Returns a sorted list of drill tools from largest to smallest i.e. reverse sort.

See SYNOPSIS for detailed usage.
