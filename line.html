SYNOPSIS

     ##################                          ##################
     ################## EXAMPLES OF SIMPLE USAGE ##################
     ##################                          ##################
     use Line;
     use Genesis;
     my $f = new Genesis;
     
     #create a vertical (r10) line 1 inch in length
     my $line = Line->new(start => [0,0], end => [0,1], size => 10);
     #draw it in the editor
     $line->draw($f);

METHODS

new

new(start => [x,y], end => [x,y], size => n_mils, polarity => 'positive')

Creates a new Line object. start is a pair of x,y coordinates for the start of the line. end is a pair of x,y coordinates for the end point of the line. size is the size (in mils) of the line (note: only round lines are available). polarity is optional, by default it is positive but can be negative as well. Additionally, attributes are not used.

draw

draw($f)

Adds the line in the current graphic editor window on the active layers. Takes a genesis object, which by convention is defined as $f. Has the default behavior as the line mode command add_line.

debugOn() debugOff()
