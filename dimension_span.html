DimensionSpan.pm

A DimensionSpan, like other objects in the drawing class, is just like a Genesis feature. The syntax to use is a little different, but similar. One difference, however, is that a the drawing object (DimensionSpan) is added in two steps. First it must be created with the new method, then it is added using the draw method. It is done in two steps because it is composed of multiple features and there is some stuff going on in the background. Additionally, this way all drawing objects can be put in a cache (such as an array) and be drawn all at once at a later time.

SYNOPSIS

     ##################                          ##################
     ################## EXAMPLES OF SIMPLE USAGE ##################
     ##################                          ##################
     use DimensionSpan;
     use Genesis;
     my $f = new Genesis;
     
     # get some dimensions and figure out the datum point for the span
     # ...
     foreach my $dim (@dimensions) {
          # make the span and draw it
          # assumes graphic editor is already open and a layer is affected
          my $span = DimensionSpan->new(origin => [$dim{x},$dim{y}],
                                        width => $dim{width}, 
                                        angle => 0, 
                                        text_size => "xl");
          $span->draw($f);
     }

METHODS

new

new(origin => [x,y], width => x_inch, angle => deg, text_size => "l")

Creates a new DimensionSpan object. The origin is the lower left of the dimension span, and it is rotated deg about the origin. The width will displayed to 4 digits. The default text_size is "l" and the available options are as follows...

option | definition

"xs" -> x,y = 30 lw = 4

"s" -> x,y = 50 lw = 6

"m" -> x,y = 80 lw = 8

"l" -> x,y = 100 lw = 10

"xl" -> x,y = 150 lw = 15

The only angles allowed are 0, 90, 180, and 270.

draw

draw($f)

Adds the dimension span in the current graphic editor window on the active layer(s). Takes a genesis object, which by convention is defined as $f.
