METHODS THAT NEED TO BE IMPLEMENTED

_init

Needs to pass on the parameters to $self

Does not return anything.

_compare

Takes two parameters.

If parameter 1 is greater than parameter 2 it returns a positive number.

If the two parameters are equal it returns 0.

Otherwise it returns a negative number.

VARIABLES THAT NEED TO BE DEFINED

_type

Defines what type of layer the sorter is capable of handling.

Can be: signal | mixed | power_ground | solder_mask | silk_screen | solder_paste | drill | rout | document.

If they type doesn't matter then set it as "any"

JOB_MOD

Requires the Job module to be set in the _init function.
