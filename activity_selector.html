ActivitySelector.pm

Greedy activity selector creates a subset S' which contains the maximum number of non-conflicting activities of the set S. The solution S' has the property where |S'| <= |S|.

SYNOPSIS

     ##################                          ##################
     ################## EXAMPLES OF SIMPLE USAGE ##################
     ##################                          ##################
     use ActivitySelector;
     use Activity;
     use Sorter;
     
     my @activities;
     for (my $i = 0; $i < n; $i++){
          push (@activities, Activity->new(activity => A, startTime = S, endTime => E));
     }
     my $selector = ActivitySelector->new(sorter => $sorter, activities => \@activities);
     my @selected = $selector->select();

METHODS

new

new(sorter => $sorter, activities => \@activities)

Constructs a new ActivitySelector.

selector is a sorter object which conforms to AbstractSorter.

activities is a list of activities which are of type Activity

select

select()

Returns an optimal subset of activities.
