SYNOPSIS

     ##################                          ##################
     ################## EXAMPLES OF SIMPLE USAGE ##################
     ##################                          ##################
     use Triangle;
     use Genesis;
     my $f = new Genesis;
     
     # create a triangle (like >) and draw it on the editor
     my $triangle = Triangle->new(origin => [0,0],
                                  width => 50,
                                  height => 100,
                                  angle => 90); # by default the polarity is positive

     $triangle->draw($f);

METHODS

new

new(origin => [x,y], width => n_mils, height => n_mils, angle => deg, polarity => 'positive')

Creates a new Triangle object. origin is the datum of the triangle symbol. angle can be 0,90,180 or 270, by default it is 0. polarity is optional, by default it is positive but can be negative if specified. Additionally, attributes are not used.

draw

draw($f)

Adds the line in the current graphic editor window on the active layers. Takes a genesis object, which by convention is defined as $f. Has the default behavior as the line mode command add_line.

debugOn/debugOff

debugOn() debugOff()

Turns off and on debugging info.
