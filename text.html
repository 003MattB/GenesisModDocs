Text.pm

This drawing object is analogous to the Genesis add_text line mode command. The end result is the same, but some of the functionality is assumed with the intention of simplifying the usage.

SYNOPSIS

     ##################                          ##################
     ################## EXAMPLES OF SIMPLE USAGE ##################
     ##################                          ##################
     use Text;
     use Genesis;
     my $f = new Genesis;
     
     # in a real use the x and y would probably be different
     my $x_loc = 0;
     my $y_loc = 0;
     
     # create some text
     # in this example angle and polarity are omitted because the defaults are fine
     my $text = Text->new(origin => [$x_loc, $y_loc],
                          size => 8,
                          line_w => 60,
                          text => "P.T.I.S.",
                          );
     # in this one we will change the angle to 270 degrees
     my $text2 = Text->new(origin => [$x_loc, $y_loc],
                          size => 8,
                          line_w => 60,
                          text => "P.T.I.S.",
                          angle => 270
                          );
                    
     #draw both of the text objects to the editor
     $text->draw($f);
     $text2->draw($f);

METHODS

new

new(origin => [x,y], size = n_mils, line_w => n_mils, text => string, angle => deg, polarity => 'positive')

Creates a new Text object. origin is a pair of x,y coordinates for the datum of the text. size is the width and height of the of the text (in mils) - it only supports having the same height and width. text is the actual text to be displayed. angle is optional, by default it is 0. polarity is optional, by default it is positive but can be negative if specified. Additionally, attributes are not used.

draw

draw($f)

Adds the text in the current graphic editor window on the active layers. Takes a genesis object, which by convention is defined as $f. Has the default behavior as the line mode command add_text.

debugOn/debugOff

debugOn() debugOff()

Turns off and on debugging info.
